Bachelor of Science in Computer Science

Credits: 120
Duration: 3 Years
Intakes: January, May, September

This comprehensive Computer Science degree provides students with a solid foundation in computing principles, programming languages, and system design. Graduates will be equipped with the skills needed for careers in software development, data analysis, and IT systems management.

Course Overview

What You Will Learn

  • Fundamentals of computer programming and algorithm design
  • Database systems and information management
  • Computer networks and cybersecurity principles
  • Software engineering methodologies
  • Artificial intelligence and machine learning concepts
  • Web and mobile application development
  • Data structures and computational theory

Program Structure

Year 1

Core Modules:
  • Introduction to Programming
  • Computer Systems Fundamentals
  • Discrete Mathematics
  • Web Development Basics

Year 2

Core Modules:
  • Data Structures and Algorithms
  • Database Systems
  • Operating Systems
  • Software Engineering

Year 3

Core Modules:
  • Artificial Intelligence
  • Computer Networks
  • Final Year Project
  • Professional Practice

Entry Requirements

  • High School Diploma with minimum B grades in Mathematics and Science
  • English language proficiency (IELTS 6.0 or equivalent)
  • Personal statement demonstrating interest in computing
  • Two academic references
